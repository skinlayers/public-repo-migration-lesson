# public-repo-migration-lesson
Fortified Cake

## Installation

Setup virtualenv and pip
```
pthon3 -m venv venv
source ./venv/bin/activate
pip install -U pip
pip install -r requirements.txt
```

Mark the script as executable
```
chmod +x main.py
```

## Usage
Activate the virtualenv, if it is not already actived
```
source ./venv/bin/activate
```

Run the script:
```
./main.py
```
